# Modelo Matemático de Dengue Hemorrágica – Tese de Doutorado

Este repositório reúne o **modelo matemático desenvolvido na minha tese de doutorado sobre dengue grave (hemorrágica) em adultos e crianças** e as principais **simulações numéricas** associadas.

## Conteúdo

- `Modelo_TESE.ipynb`  
  Implementação do **modelo geral** em equações diferenciais ordinárias, com diferentes métodos numéricos e geração de figuras.

- `Analise_de_Sensibilidade_R0.ipynb`  
  Cálculo do **número reprodutivo básico $\(\mathcal{R}_0\)$** e **análise de sensibilidade global** usando eFAST/FAST99.

- `README_do_Modelo.md`  
  Descrição detalhada do modelo geral e das simulações.

- `README_R0.md`  
  Descrição da formulação de $\(\mathcal{R}_0\)$ e da análise de sensibilidade.

## Objetivo

Fornecer uma base clara para **reproduzir, estudar e estender** o modelo, conectando diretamente o código aos resultados apresentados na tese.
